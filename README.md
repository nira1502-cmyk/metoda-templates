# Metoda Templates

תבניות מסמכים ממותגות לנירה טל | מתודה פתרונות AI

## קבצים

| קובץ | סוג | שימוש |
|------|-----|--------|
| `guide-template.html` | מדריך למשתמש | מסמך הסבר / How-to |
| `spec-template.html` | אפיון | מסמך דרישות לפרויקט דיגיטלי |
| `proposal-template.html` | הצעת מחיר | הצעה מסחרית ללקוח |
| `shared-styles.css` | עיצוב | CSS משותף לכל התבניות |

## שימוש

1. העתיקו את קובץ התבנית המתאים
2. החליפו את כל ה-`[...]` בתוכן האמיתי
3. הוסיפו `logo.png` ו-`favicon.png` לאותה תיקייה
4. פתחו בדפדפן או העלו ל-Vercel
